# Nand2Tetris — 16-bit Computer from Scratch

A complete 16-bit computer system built from the ground up, following the Nand2Tetris course (nand2tetris.org).

---

## What's inside

- **Project 1** — Basic logic gates (AND, OR, NOT, MUX, DMUX, etc.) in HDL
- **Project 2** — Arithmetic chips: Adders and ALU
- **Project 3** — Sequential chips: Registers, RAM, Program Counter
- **Project 4** — Assembly programs (Fill, Mult)
- **Project 5** — Full computer chip: CPU, Memory, Computer
- **Project 6** — Assembler that translates Hack assembly into binary

---

## Technologies

`HDL` `Assembly` `Hack Architecture`

---

## About the course

Nand2Tetris (also known as "From NAND to Tetris") is a project-based course that teaches how computers work from the ground up — starting with a single NAND gate and building all the way up to a working computer system.

[nand2tetris.org](https://www.nand2tetris.org)
